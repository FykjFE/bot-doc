## 免安装初始化
找一个自己喜欢的文件夹，执行以下命令
```bash
$ npx @fykj/bot init my-app

$ cd my-app

$ npm install

$ npm start
```

:::tip
我们建议你使用npx初始化项目
:::
## 全局安装
```bash
$ yarn global @fykj/bot

# or

$ npm install @fykj/bot -g

# 初始化

$ bot init my-app
```

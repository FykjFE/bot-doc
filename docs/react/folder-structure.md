title: 目录结构
---

创建后，您的项目应如下所示:
```bash
├─config #webpack配置文件
├─public #公共文件夹
├─script #脚本命令
└─src
    ├─assets #静态资源
    │  ├─img #图片资源
    │  └─svg #svg图片
    ├─components #公共组件
    ├─hooks #封装的hooks
    ├─interfaces #ts定义的文件
    ├─pages #容器组件
    ├─routes #路由
    ├─services #api统一管理
    ├─store #sotre管理
    │  ├─actions
    │  ├─constants
    │  └─reducers
    ├─styles #样式管理
    └─utils #工具类
```

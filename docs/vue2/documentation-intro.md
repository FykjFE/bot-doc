```bash
npm install
npm run  serve
```

> 全局注册组件

`src/components`目录下组件已自动全局注册，使用大驼峰命名。

```javascript
#bad
hello-world.vue

#good
HelloWorld.vue
```
使用时使用连字符
```javascript
<hello-world></hello-world>
```
